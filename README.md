# Android OTP Assignment

## OTP Logic
- 6 digit OTP generated locally
- Expires in 60 seconds
- Max 3 attempts
- New OTP invalidates old

## Data Structure
- Map<String, OtpData> used for storing OTP per email

## External SDK
- Timber used for logging analytics events

## AI Usage
- GPT used for guidance and structure
